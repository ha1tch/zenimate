sinclair.bdf
ZX Spectrum 48K character set

This BDF face contains the 8x8 glyph shapes of the Sinclair ZX Spectrum 48K
character set (codes 32-127). It is bundled from the ha1tch/bdf-fonts catalogue:
https://github.com/ha1tch/bdf-fonts

In that catalogue, sinclair.bdf is the reproducible result of extracting the
character set from a ZX Spectrum 48K ROM. The same supporting files are bundled
here in the sinclair/ subdirectory:

  sinclair/zx48.rom      the ZX Spectrum 48K ROM the font was extracted from
  sinclair/sinclair.go   the extraction program (the author's own work, placed
                         in the public domain)
  sinclair/SINCLAIR.txt  the copyright/permission statement (reproduced below)
  sinclair/amstrad.txt   Amstrad's permission, as issued on the company's
                         behalf by Cliff Lawson

The glyph data in sinclair.bdf is byte-identical to the catalogue's, and is
reproducible from the bundled ROM via the bundled extraction program.
